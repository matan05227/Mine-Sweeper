# Mine sweeper

A simple browser-based Pacman exercise submission for Lesson 12.

**Quick Start**

- Open the game by double-clicking `index.html` in your browser.
- Or serve the folder with a simple local server and open http://localhost:8000:

```bash
python -m http.server 8000
```

**Controls**

- Use the arrow keys to move the player (Pacman) around the grid.

**Project Structure**

- [index.html](index.html) — game entry HTML
- [css/style.css](css/style.css) — styles
- [js/game.js](js/game.js) — game logic

**Development**

- Edit `js/game.js` to change behavior and game rules.
- Edit `css/style.css` to update visuals and layout.

**Notes**

- This is a student exercise build; behavior and features may be minimal.
